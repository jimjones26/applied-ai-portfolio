04 / Core Systems

Engineering at the
frontier of trust.

Each subsystem purpose-built from first principles. No third-party agent SDKs. No SaaS databases. Every component shares the Chronicle. State survives everything.

SYS:01
Cryptography

Cryptographic
Sovereignty Layer

Hardware-backed root keys tied to TPM and Secure Enclave. Envelope encryption for every artifact. Zero-plaintext guarantee: no data touches persistent storage or wire without encryption.

  • AES-256-GCM at rest and in transit
  • Ed25519 + ML-DSA-65 hybrid signing
  • X25519 + ML-KEM-768 post-quantum key agreement
  • Argon2id passphrase derivation + HKDF-SHA256
  • Shamir secret sharing for key recovery
SYS:02
Persistence

Chronicle Fabric

Single source of truth. Append-only encrypted binary log with cryptographic ordering. Every event — user input, memory ops, tool calls, key rotations — is immutable and auditable.

  • 64MB segment-based log + checkpoint snapshots
  • Full system replayability with timeline scrubber
  • Tiered fsync: critical immediate, routine batched
  • mmap projection system targeting sub-100ms recall
SYS:03
Cognition

Neuron Runtime

Lightweight actor model with recursive delegation to arbitrary depth. Supervision trees with configurable policies. Full observability via trace IDs and ancestry chains.

  • Isolated execution units with private mailboxes
  • Recursive delegation — default max depth: 5
  • RestartWithBackoff, EscalateToParent, PruneBranch
  • 60-second checkpointing for long-running tasks
SYS:04
Autonomy

Intelligence Routing

The organism owns the mind; it rents neurons. Sovereignty-first routing prefers local execution. Distillation loop converts every external call into training data toward self-sufficiency.

  • Capability profiling: task fit, reliability, sovereignty
  • Hybrid routing: local subtasks + minimal escalation
  • Distillation loop toward local model independence
  • Ethical alignment evaluation on every routing decision
